⚔️ Пир Нартов (Feast of the Narts)
«Пир Нартов» — это соревновательная карточная стратегия в сеттинге осетинского нартского эпоса. Вдохновленная механикой классических ККИ, игра предлагает игрокам сесть за легендарный стол нартов и доказать свое право на звание величайшего героя, используя тактику, магию и силу своего рода.

📜 Сюжет и Атмосфера
На великий пир собрались три главных рода: Ахсартаггата, Бората и Алагата. Каждый хочет занять почетное место у Чаши Уацамонга. Но место достанется лишь тому, чья армия на игровом столе окажется самой доблестной.

🛠 Технологический стек
Core: ASP.NET Core 8 (MVC)

Real-time: SignalR (синхронизация ходов)

Database: Entity Framework Core (хранение профилей, статистики и колод)

Frontend: JavaScript / CSS Grid (или React для UI *еще не решили*)

🎲 Геймплейные особенности
Три раунда: Победи в двух, чтобы забрать славу.

Ряды боя: * Ближний бой — доблестные воины и стальные герои.

Дальний бой — охотники и легендарные стрелки.

Осада — великаны-уаиги и магические силы природы.

Чаша Уацамонга: Уникальная механика проверки «чести». Особая карта, которая может как возвысить твой отряд, так и разоблачить хитрость врага.
